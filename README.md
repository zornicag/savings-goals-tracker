# Savings Goals Tracker

Savings Goals Tracker is a Spring Boot web application for creating and managing personal savings goals.

The application allows users to register, log in, create savings goals, add deposits, withdraw money when needed, and track their progress.

## Technology Stack

* Java 17
* Spring Boot 3.4.0
* Spring MVC
* Thymeleaf
* Spring Data JPA
* MySQL
* Maven
* HTML
* CSS

## Main Functionalities

* User registration
* User login
* Create savings goals
* View savings goals
* Add deposits
* Withdraw money
* Track goal progress

## Domain Entities

* User
* Category
* SavingsGoal
* Transaction
